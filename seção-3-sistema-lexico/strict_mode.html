Strict Mode - JS

  • Impede que variáveis sejam criadas sem serem declaradas
  • O "use script" deve sempre ser a primeira linha no script para ativá-lo
  • Saua principal intenção de uso é deixar o código mais seguro
  • O use strict pode ser usado em um código inteiro, ou esepcificamente em uma função
  • O use strict não permite que dois parâmetros por exemplo tenham o mesmo nome de declaração(ver console)
